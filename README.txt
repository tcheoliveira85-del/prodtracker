# ProdTracker PWA

Arquivos incluídos:
- `index.html`
- `manifest.webmanifest`
- `sw.js`
- `icons/`

## Como publicar
Para funcionar como PWA instalável, hospede estes arquivos em HTTPS.
Opções simples:
- GitHub Pages
- Netlify
- Vercel
- servidor interno com HTTPS

## Como instalar
### Android / Chrome
Abra o site e toque em **Instalar app**.

### iPhone / Safari
Abra o site, toque em **Compartilhar** e depois em **Adicionar à Tela de Início**.

## Observação
Abrir o `index.html` direto pelo celular via `file://` não ativa instalação nem service worker.
